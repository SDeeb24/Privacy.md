# Privacy Policy

**Last updated: June 18, 2026**

## Overview

Mr. Popcorn Reacts Auto-Poster is a personal tool used exclusively to generate and post content to a single TikTok account. It is not a public product and is not available to other users.

## Data Collection

This app does not collect, store, or process any personal data from TikTok users. The only data used is the access token and open ID of the authorized TikTok account owner, which are stored locally on the operator's device and never shared with third parties.

## TikTok API Usage

This app uses the TikTok Content Posting API solely to upload videos and photos to an authorized TikTok account. No data is read from TikTok users' profiles beyond what is required to complete the post.

## Third-Party Services

This app uses Higgsfield AI to generate video and image content. Please refer to Higgsfield's privacy policy for information on how they handle data.
